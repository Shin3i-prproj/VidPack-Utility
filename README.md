# 🎬 VidPack Utility

**Fast • Clean • Powerful**

VidPack Utility is a command-line video compression tool built with Python and FFmpeg. It makes compressing videos simple while providing useful features like batch processing, remembered settings, compression logs, and real-time progress tracking.

VidPack Utility started as a learning project to explore Python, FFmpeg, and software architecture. As development progressed, it evolved into a modular command-line application featuring batch processing, logging, persistent settings, and a clean, maintainable codebase.

---

## ✨ Features

- 🎥 Compress a single video
- 📁 Batch compress an entire folder
- ⚡ Multiple compression presets
- 💾 Remembers your last selected preset
- 📂 Remembers your preferred output folder
- 📊 Displays compression statistics
- 📝 Automatically saves compression logs
- ⚙️ Built-in settings menu
- ℹ️ About page with version information

---

## 🛠️ Technologies Used

## 🚀 Getting Started

### Requirements

- Python 3.11 or newer
- FFmpeg installed and available in your system PATH

### Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/vidpack-utility.git
cd vidpack-utility
```

### Run the application

```bash
python launcher.py
```
